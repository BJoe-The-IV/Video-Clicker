# Video-Clicker
A fun clicker game to play while watching YouTube.

## How To Use
First, visit https://the-bjoe.github.io/Video-Clicker/

Then, simply copy and paste a YouTube link into the search bar at the top of the page and press enter. Now, you can click the button to play the YouTube video.
